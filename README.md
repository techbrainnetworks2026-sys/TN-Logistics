# Techbrain Networks Logistics Website

A responsive React website for Techbrain Networks, showcasing intelligent logistics solutions.

## Features

- Fully responsive design (Desktop, Tablet, Mobile)
- Sticky navbar with shadow on scroll
- Smooth scrolling navigation
- Mobile hamburger menu
- Reusable product and service cards
- Grid layouts
- Hover animations
- Fade-in animations on scroll
- Modern UI with logistics-themed color palette

## Tech Stack

- React
- CSS (Flexbox + Grid)
- Vite

## Installation

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server

## Usage

The website is a single-page application with sections for Home, About, Team, Products, Industries, Services, Contact, and Careers.

Images are placeholders; replace with actual 4K images generated from the provided prompts.

## Color Palette

- Primary: #0D47A1 (Deep Logistics Blue)
- Secondary: #1976D2 (Tech Blue)
- Accent: #64B5F6 (Light Blue)
- Highlight: #00C853 (Logistics Green)
- Background: #F4F8FB (Soft white-blue)
- Text: #0A1929 (Dark tech)

## Image Prompts

All image alt texts contain the 4K image generation prompts for backgrounds and sections.
